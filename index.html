FAQ

Common questions.

Is it safe for games with anti-cheat?

Yes. GammaBoost+ calls the standard Windows gamma ramp API — the same path the OS Night Light feature uses. No kernel drivers, no process injection, no overlay hooks, no game-memory access. It operates strictly at the display-pipeline level.

Which games does it work with?

Every game. Because the boost is applied at the OS level, every pixel reaching your monitor goes through it — regardless of engine, fullscreen mode, or anti-cheat system.

What are the system requirements?

Windows 10 or 11 (64-bit). Any modern GPU. 200 MB of storage. Steam required for achievements, cloud sync, and community configs — core gamma features work without it.

Can I use it while streaming on OBS?

Yes. GammaBoost+ connects to OBS via WebSocket and applies a matching color filter so viewers see the same boosted image. Export .cube LUTs for DaVinci, Premiere, or any NLE.

What languages are supported?

Ten: English, Spanish, French, German, Portuguese, Japanese, Simplified Chinese, Korean, Russian, and Turkish.

Does it work with multiple monitors?

Yes. Pick which display to adjust, or apply globally. Works with ultrawides and mixed-refresh-rate setups — anything that exposes a standard gamma ramp.
